# Hello Command

Greet the user with a friendly message.

If a name is provided via `$1`, use it in the greeting. Otherwise, use a generic greeting.

## Instructions

1. If `$1` is provided, greet the user by name: "Hello, $1!"
2. If no name is provided, say: "Hello from the HumanInLoop marketplace!"
3. Briefly explain that this is an example plugin command.
